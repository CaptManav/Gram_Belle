# Gram_Belle
A personality‑driven Android AI voice assistant that listens, thinks, and talks back — fast. Powered by Groq (Whisper + LLaMA 3) for real‑time reasoning and XTTS for human‑like neural speech. Built with Kotlin + FastAPI, it’s a sharp, sassy AI tutor that explains, challenges, and roasts — all through a full end‑to‑end voice pipeline.
